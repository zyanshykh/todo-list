# 🗺️ Modern Todo List

A sleek, intuitive, and modern task management application built to streamline your daily workflow.

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg?style=flat-square)](http://makeapullrequest.com)

---

## 🚀 Features

*   **Beautiful UI:** Modern and responsive design with dark/light mode support.
*   **Smart Categorization:** Group your tasks using tags, priorities, or due dates.
*   **Persistent Storage:** Your tasks stay saved even after refreshing the page.
*   **Performance First:** Fast, lightweight, and dependency-free core logic.

---

## 🛠️ Tech Stack

*   **Frontend:** HTML5, CSS3 (Tailwind CSS), JavaScript (ES6+)
*   **Storage:** LocalStorage / API Integration

---

## 📦 Getting Started

Follow these simple steps to get a local copy up and running.

### Prerequisites
Make sure you have a modern web browser installed.

### Installation

1. Clone the repository:
   ```bash
   git clone [https://github.com/zyanshykh/todo-list.git](https://github.com/zyanshykh/todo-list.git)
   ```
   ## Navigate to the project directory:
   ```
   cd todo-list
   ```
